# 문제

### 추상 클래스 활용하기: `Vehicle` (탈것)

`Character` 클래스 예제를 참고하여 다음 조건을 만족하는 코드를 작성해보세요.

1. **추상 클래스 `Vehicle` 생성**:
    - 필드: `String modelName` (모델명), `int speed` (현재 속도)
    - 생성자: 모델명을 받아 초기화하고, 속도는 0으로 설정
    - 일반 메서드: `accelerate(int amount)` (속도 증가)
    - **추상 메서드**: `move()` (이동 방식 출력)
2. **자식 클래스 `Car`, `Bicycle` 생성**:
    - `Vehicle`을 상속받습니다.
    - `move()`를 오버라이딩하여 각자의 이동 방식을 출력합니다. (예: "도로를 달립니다", "자전거 도로를 달립니다")
3. **Main 클래스**:
    - `Vehicle` 타입의 배열에 `Car`와 `Bicycle` 객체를 담고 반복문을 통해 `move()`를 호출하세요.
