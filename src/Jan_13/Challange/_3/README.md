# 문제

### 문제: 카페 키오스크 시스템 (추상 클래스 & 상수)

카페에서 판매하는 음료를 관리하는 시스템입니다. 모든 음료는 공통적인 특징을 가지지만, 가격을 계산하는 방식은 다를 수 있습니다.

- **요구사항:**
    1. `CafeConstants` 클래스에 **상수**로 `CAFE_NAME`("자바 카페"), `SIZE_SMALL`(1), `SIZE_LARGE`(2)를 정의하세요.
    2. `Drink` **추상 클래스**를 만드세요.
        - 필드: `name` (이름), `size` (사이즈)
        - 생성자: 이름과 사이즈를 초기화
        - 추상 메서드: `calculatePrice()` (가격 계산 로직)
        - 일반 메서드: `info()` ("자바 카페의 [이름] 음료입니다." 출력)
    3. `Americano` 클래스는 `Drink`를 상속받습니다.
        - 기본 가격 3000원, 사이즈가 `SIZE_LARGE`라면 500원 추가.
    4. `Main`에서 `Americano` 객체를 생성하고 `info()`와 계산된 가격을 출력하세요.
- **출력 예시:**
    
    ```java
    === 자바 카페 ===
    자바 카페의 아메리카노 음료입니다.
    가격: 3500원
    ```
