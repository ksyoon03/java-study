# 문제

### **문제: 싱글톤 패턴 구현 및 래퍼 클래스 활용**

사용자의 접속 수를 카운트하는 `UserCounter` 클래스를 싱글톤으로 구현하세요.

- **요구사항**:
    1. `UserCounter` 클래스는 프로그램 내에서 단 하나만 생성되어야 합니다 (`getInstance` 활용).
    2. `count`라는 `Integer` 타입의 필드를 가지며 초기값은 0입니다.
    3. `visit()` 메서드를 호출할 때마다 `count`가 1씩 증가해야 합니다 (오토박싱/언박싱 활용).
    4. `getCount()` 메서드로 현재 카운트를 반환하세요.
- **출력 예시**:
    
    ```java
    현재 접속자 수: 1
    현재 접속자 수: 2
    user1과 user2는 동일한 객체인가요? true
    ```
