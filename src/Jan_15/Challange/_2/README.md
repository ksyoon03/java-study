# 문제

### **문제: 빌더 패턴 직접 구현하기 (정적 내부 클래스)**

`Laptop` 클래스를 정의하고, 빌더 패턴을 사용하여 객체를 생성하는 코드를 작성하세요. (Lombok 사용 금지)

- **요구사항:**
    1. `Laptop` 클래스는 `model`(String)과 `price`(int)를 필드로 가집니다.
    2. `Laptop` 내부에 `static class Builder`를 정의하여 메서드 체이닝을 지원해야 합니다.
    3. `main` 메서드에서 빌더를 사용해 모델명 "MacBook", 가격 200 객체를 생성하고 정보를 출력하세요.
- **출력 예시:**
    
    ```java
    노트북 모델: MacBook, 가격: 200
    ```
